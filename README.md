# Pi-Sensors

## Setup

1. Run `make setup`,
1. Open `hosts.yml` in your text editor and complete the `hosts` and `vars`
   sections according to your setup.

## Run live server

Run `make serve`

## Deploy to Raspberry Pi

Run `make deploy`
