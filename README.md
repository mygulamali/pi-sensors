# Pi-Sensors

## Setup

1. Run `make init`,
1. Open `ansible/hosts.yml` in your text editor and complete the `hosts` and
   `vars` sections according to your setup,
1. Run `make setup`.

## Run local development server

Run `make serve`

## Deploy to Raspberry Pi

Run `make deploy`
